TextPlayer Commands

THIS IS OUT OF DATE!

The TextPlayer engine (Tardproof / Lucidity) lets you put in plain text and it transforms it to rich html. It uses Tokens in the text, which are commands encapsulating values that are transformed in a particular way. In general, a token is written with curly brackets around it, and encapsulates the text or image within the brackets. Multiple values are supported and in cases required, and these are each seperated with a vertical bar character - |. This document lists the supported tokens:


{link My Link|pageName.html}

Link to a different page within the Content folder.

<a
href=”Content/pageName.html”>My Link</a>


{external My Link|http://abc.net.au}

Link to an external site, uses an icon to denote the link is external.

<a
target='_blank' href='
http://abc.net.au
'><i
class='icon-external-link'></i> My Link</a>


{load filename.html}

Load the content of an external file into the place where the token is

Output: (contents of file between <body> tags)

Note: If filename contains :// then it tries to load it from an external source, otherwise it tries to load the file from the Contents/ folder.


{parse filename.html}

Output: (contents of file between <body> tags)

Similar option to {load} except that it will look for and interpret tokens in the external file.


{overlay Show stuff|stuff.html}
{popup Show stuff|stuff.html}

Draws a button on the page with the text that links to an overlay window. See the naming help on the Pages tab for how you can name includes so they are parsed properly.

<a
href='Content/stuff.html' rev='overlay'
class='rp-button-dialogue'>Show stuff</a>


{centered show this text in the middle}
{center show this text in the middle}

Draws the value centered on the page. Often used to wrap an overlay button.

<p
class='centered'>show this text in the middle</p>


{fastfact Quick Note!|In australia, the sky is blue just like in America.}

Draws a fast-fact box on the right of the page. If no text is entered before the first | character, the words “fast fact” are used.

<div
id='fastfact'><h3>Quick Note!</h3><p>In
australia, the sky is blue just like in America.</p></div>


{image foo.jpg}

Draws an image from the /Content/media folder

<img
src='Content/media/foo.jpg' class='rp-image' />


{bullets point 1|point 2|point 3|point 4}

Draws a list of dot points

<ul><li>point
1</li><li>point 2</li><li>point
3</li><li>point 4</li></ul>


{numbers point 1|point 2|point 3|point 4}

Draws a numbered list of points

<ol><li>point
1</li><li>point 2</li><li>point
3</li><li>point 4</li></ol>


{rightimages image1.jpg|image2.png|image3.gif}

Draws images on the right of the page that fade in in sequence when the page loads. Images are assumed to be in the /Content/media folder. Supports one or more images. If the first element does not end in an extension, it is assumed to be a classname, which is applied to each image.

<img
src='Content/media/image1.jpg' class='rp-fadein' /><img
src='Content/media/image2.png' class='rp-fadein' /><img
src='Content/media/image3.gif' class='rp-fadein' />


{tag marquee|i hate the marquee tag}

Draws any html tag to the page. Useful because all HTML is stripped to begin with. Can be useful for adding things like bold or italic text. Can not specify any attributes.

<marquee>I
hate the marquee tag</marquee>


{ref 6}

Draws a number within a super tag. Javascript then automatically links the number to the matching value in the resources.html file (assumes ordered list is used for references).

<sup>6</sup>


{term definition of a strawberry|strawberry}

Draws a link to a definition (uses glossary.html, requires each term to have an anchor named the same as the value in the data attribute).

<dfn
data='strawberry'>definition of a strawberry</dfn>


{clickimage image1.jpg|A picture of a duck|image2.jpg|A picture of a banana}

Draws a sequence of images with a checkbox over the right top corner that allows the user to click to select one or more items. Stores the values in the SCO. The text value of each image is revealed in feedback after the user clicks Submit or re-loads an already saved page.


{clickcheck I feel great|Ifeel ok|I feel terrible}

Draws a sequence of checkboxes with text labels to the right that allows the user to click to select one or more items. Stores the values in the SCO.


{survey 5|Grapes are tasty|Apples are nice|Bananas are wonderful}

Draws a series of radio buttons for each line of text. Useful for surveys where one or more texts have a rating from 1 to N. The first parameter determines the number of radio buttons to draw for each line. Stores results in the SCO.


{wrap spice|salt, pepper, tumeric}

Draws a span tag with the specified class wrapping the content.

<span
class='spice'>salt, pepper, tumeric</span>


{tabs Bananas|bananas.html|Apricots|apricots aren't too bad either}

Draws a tab strip and loads or inserts content for each tab. When loading content, it assumes the files are stored in the Content/ folder and can't load from external sources. Does not parse the resulting files, and only shows the value within the <body> tags. Each set of two parameters has first the title, then the url. If the second parameter doesn't contain a filename, the value becomes the body text within the tab itself.


{quiz}

Will draw a quiz; not yet functioning.

{anything else}

If the tag isn't matched, it is drawn as a standard paragraph within <p> tags.
